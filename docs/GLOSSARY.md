# Glossary

| Term | Meaning |
|---|---|
| MRO | Maintenance, Repair, and Operations |
| CMMS | Computerized Maintenance Management System |
| EPS | Equipment Passportization System |
| MMS | Maintenance Management System |
| WMS | Warehouse Management System |
| SRS | Service Request System |
| WO | Work Order |
| PM | Preventive Maintenance |
| CR | Change Request |
| RBAC | Role-Based Access Control |
| SSO | Single Sign-On |
| LDAP | Lightweight Directory Access Protocol |
| AD | Active Directory |
| SLA | Service-Level Agreement |
| BOM | Bill of Materials |
| Audit Event | Immutable record of a meaningful system action |
| Source of Truth | The owning system/module for a type of data |
| DTO | Data Transfer Object |
| Entity | Persistence model mapped to database table |
| Projection | Read-optimized representation of data |
| Modular Monolith | Single deployable app with strong internal module boundaries |
